## Main Stack
Project is created with:
- Vue.js
- Vuetify
- Typescript

## Getting Started
### Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```
